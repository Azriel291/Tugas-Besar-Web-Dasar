Timeline Sejarah

Perjalanan Kota Bandung

1810

Bandung Resmi Didirikan

R.A. Wiranatakusumah II mendirikan pusat pemerintahan baru Bandung atas perintah Herman Willem Daendels.

1884

Jalur Kereta Api Hadir

Kehadiran jalur kereta api mempercepat pertumbuhan ekonomi dan menjadikan Bandung lebih mudah diakses.

1920-an

Paris van Java

Bandung dikenal sebagai Paris van Java karena keindahan arsitektur, pusat mode, dan gaya hidup modern.

1955

Konferensi Asia-Afrika

Bandung menjadi pusat perhatian dunia melalui Konferensi Asia-Afrika yang dihadiri 29 negara.

Sekarang

Kota Kreatif Indonesia

Bandung berkembang menjadi pusat pendidikan, teknologi, pariwisata, dan industri kreatif.

Pesona Bandung

Ruang kecil untuk mengenal budaya, rasa, dan cerita Kota Kembang.
